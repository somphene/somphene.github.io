### Language of Mathematics

### Abstract
Having introduced a facing number approach, please may we introduce an Exchanger. This standard notion for this is a Function.

### Function
There is a direct facing way so we do not have identity function or inverse functions. The top number is used to make general statement. Let us have $$9999$$ as a top number so that $$f(x)= x^2$$ is now an exchanger $$9999 \to 9999\cdot 9999$$. Exchanger allows each input to be exchanged so that $$f(2) = 2^2$$ is now $$2 \to 2^2$$.

The function $$f(x) = x+1$$ is now $$9999 \to 9999 +1$$.

The top number is chosen to be greater than any number inlvolved in the process and hence is absoultely supreme making it a ddistinguished number.

An example of multivarivable function is now an exchanger: $$f(x,y)= x+y$$ is now the exchanger: $$(9999_1, 9999_2) \to 9999_1 +9999_2$$.
